## üöÄ Overview

**ClinFlow Insight** is a next-generation clinical trial data monitoring platform designed for high-velocity research environments. It integrates real-time data flow visualization with AI-powered insights to detect anomalies, optimize data quality, and streamline regulatory compliance.

Built with a **modern glassmorphism aesthetic** and **dark mode** by default, it offers an immersive, executive-grade dashboard for clinical data scientists and trial managers.

- **üß† AI-Powered Analysis**: Integrated **Google Gemini AI** to automatically analyze data nodes, explain anomalies, and suggest remediation steps in real-time.
- **üåä Real-Time Data Flow**: Interactive node-based visualization of clinical data pipelines (EDC, CTMS, Labs).
- **üõ°Ô∏è Quality Gatekeepers**: Visual status indicators for data integrity checks (Green/Red/Amber).

## ‚ö° Quick Start

### Prerequisites

- Node.js 18+
- npm or pnpm
- A Google Gemini API Key

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/HVVSATHWIK/clinflow-insight.git
    cd clinflow-insight
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env.local` file in the root directory:
    ```env
    GEMINI_API_TOKEN=your_api_key_here
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```

## üöÄ Deployment

### Deploy to Vercel

The project is optimized for deployment on [Vercel](https://vercel.com).

1.  Push your code to a Git repository (GitHub/GitLab).
2.  Import the project into Vercel.
3.  Add the `GEMINI_API_TOKEN` to the Vercel Project Settings > Environment Variables.
4.  Deploy!
